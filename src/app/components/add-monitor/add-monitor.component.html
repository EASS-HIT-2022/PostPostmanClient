Add new Monitor

In order to use postpostman monitors you should create a new colection of HTTP requests via Poastman.

1. Download Postman and open the app.

2. Create a new postman collection.

3. Add HTTP requests to the created collection, that will test your app liveness and business logic.

4. Click Share and choose 'Via JSON link' option.

5. Copy the created link and use it in the form field (collection url).
